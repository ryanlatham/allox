# Codex Project Notes

This project was generated by `allox` for a Codex-led workflow.

- Codex is the only visible runtime during normal project development.
- Task and review orchestration happens through `scripts/ai/*.py`, which are thin shims over the installed `allox` CLI.
- Worktree setup helpers live behind `scripts/ai/worktree_setup.py` and `allox project worktree-setup`.
- Shareable project-local Codex environment files belong under `.codex/` when needed.

Run `python3 scripts/ai/doctor.py` from the project root to inspect the local environment from inside the project.
